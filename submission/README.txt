Implemented Harris detector, SIFT descriptor and matching, also ANMS.

Wrote and tested under Visual Studio 2017. Binary execution file can be found under ./bin folder.

Usage: SIFT.exe [path to image 1] [path to image 2] [threshold for corner detection(suggestion: 80)] [threshold for matching(suggestion: 5)] [flag to use ANMS: 0-no, 1-yes]


Other features:
1. Rotation invariance 
2. Contrast invariance
3. Scale invariance to some degree
